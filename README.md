# Slanted View — Production Packet 1

This packet is the first uploadable production foundation for slantedview.com.

## Included
- Cinematic responsive homepage matching the approved proof
- Approved SV eye logo asset
- Data-driven Trending, Latest, Shorts, and Podcast sections
- Responsive mobile navigation
- Search dialog shell
- Newsletter interaction shell
- SEO metadata, manifest, robots.txt, sitemap.xml
- GitHub Pages `.nojekyll` file

## Upload to GitHub
1. Extract this ZIP on the Chromebook.
2. Open the repository: `slantedviewent-jpg/slantedview.com`
3. Choose **Add file → Upload files**.
4. Drag ALL CONTENTS of this extracted folder into GitHub.
5. Confirm the folders remain intact:
   - assets/
   - css/
   - data/
   - js/
6. Commit to the `main` branch.
